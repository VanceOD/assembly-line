# Build-a-Toy Workshop

## Summary

This is the source code for [Gamedev.js Jam 2026](https://itch.io/jam/gamedevjs-2026). Normally I keep my repositories private because I'm not particularly proud of my coding skills, but one of the challenges for this game jam is the [Open Source Challenge](https://gamedevjs.com/jam/2026/#challenge-opensource). So I figured, why not. And if anyone finds this code helpful in any way, then that's a plus.



Originally I was not going to enter the open source challenge. And I wanted to build something quickly without staying up late hours. So the code is, somewhat intentionally, rushed and unorganized (even for me). It's a small and simple project though, so I think that can be forgiven.



### Play the Game

If you just want to play the game, you can [play the web version here](https://rythim-man.itch.io/build-a-toy-workshop). You can also click the **Download Now** button to download executables for Windows and Android. Executables use Forward+ rendering, which runs a little faster and looks a little nicer than the web version.



Note: On Android, you will need to give your phone permission to install apps outside the app store. This process is different for every vendor, so you may need to Google search how to do it for your particular device.



### Cross-Platform Support

This game was written in gdscript using the Godot 4.6.1 game engine. This means that anyone with Godot downloaded on their system should be able to easily export the game to any platform (except console) by cloning this repository. I don't have a Mac, or a PC with an ARM chip to test it on those platforms though. If you want executables for those systems you'll need to build it yourself.
